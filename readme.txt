Funcionalidades personalizadas modularizadas para la academia.

== Description ==

Este plugin integra funcionalidades avanzadas para Origin LMS, incluyendo:
*   Sistema de CRM y gestión de permisos.
*   Dashboard financiero y registro de pagos.
*   Sistema de gamificación y tracking de tiempo.
*   Motor de email marketing con plantillas.
*   Sistema de logros y notificaciones.
*   Mejoras visuales para LearnDash (Sliders, Grids).

== Installation ==

1. Sube la carpeta `origin-lms` al directorio `/wp-content/plugins/`.
2. Activa el plugin desde el menú 'Plugins' en WordPress.
3. Configura los shortcodes en las páginas correspondientes.
